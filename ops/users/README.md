# Manage users in SFENGINEERING.AIFDE account

**Instructions**

Example usage:

```sh
./define_user.sh
```

- Run this first to create the YAML configuration for a new user

```sh
CONNECTION=aifde.setli ./create_user.sh setli
```

- Replace `aifde.setli` with the Snow CLI connection name you would like to use.
- Replace `setli` with the user you would like to create
