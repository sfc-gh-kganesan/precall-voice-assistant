

# Balto Components

[![GitHub branch status](https://github.com/snowflakedb/balto/actions/workflows/storybook.yaml/badge.svg)](https://github.com/snowflakedb/balto/commits/HEAD/)

[Balto Components](https://balto.snowflake.com/) is a library of React components that implements the Balto Design System.
It is used in SnowSight and other applications and aims to be rigorously battle-tested through multiple use-cases to ensure re-usability and portability for use in new applications.

Get started in the [Balto documentation](https://balto.snowflake.com/).
