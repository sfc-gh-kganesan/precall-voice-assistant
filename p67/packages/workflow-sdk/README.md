# @p67/workflow-sdk

Workflow SDK for the P67 platform.

## Installation

```bash
pnpm install @p67/workflow-sdk
```

## Usage

```typescript
import { version } from "@p67/workflow-sdk";

console.log(version);
```

## Development

```bash
# Install dependencies
pnpm install

# Development mode (watch)
pnpm dev

# Build (TypeScript compilation)
pnpm build

# Bundle (single file with all dependencies)
pnpm bundle

# Lint
pnpm lint
pnpm lint:fix

# Format
pnpm format
pnpm format:check

# Type check
pnpm type:check

```
