| Framework | Description | Signals |
|---|---|---|
| `streamlit` | Streamlit apps and components | st., streamlit, pages/, .streamlit/ |
| `dbt` | dbt models, macros, packages | ref(), source(), dbt_project.yml, macros/ |
| `cortex` | Cortex AI functions and features | CORTEX., AI_COMPLETE, AI_EXTRACT, AI_CLASSIFY |

| `spcs` | Snowpark Container Services | Dockerfile, service spec, compute pool, image repository |
| `native-app` | Snowflake Native Apps | setup.sql, manifest.yml, application package |
| `react` | React and Next.js frontend apps | React, useState, useEffect, JSX, TSX |
| `flask` | Flask web applications | Flask, @app.route, Blueprint |
| `fastapi` | FastAPI web services | FastAPI, @app.get, Pydantic, uvicorn |
| `langchain` | LangChain / LangGraph orchestration | LangChain, LangGraph, ChatModel, chain |
| `pdfplumber` | PDF table extraction and text parsing | pdfplumber, extract_tables, page.chars, page.lines |
